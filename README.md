# A Minecraft tools library, written in Python

This library can be used to:
 - print a cartography based on the in-game crafted maps
 - manage the server
 - manage the whitelist
 - manage the banned players

## Requirements

 - Python v3.5
 - Pillow library
 - NBT library (included for now)
